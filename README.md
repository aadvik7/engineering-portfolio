## Project Highlights

### 01 — Ecommerce Sales Analyzer
End-to-end pipeline on the Brazilian Olist dataset (~100k orders). Merges 9 CSVs, cleans data with Pandas, loads to PostgreSQL, runs 7 analytical SQL queries, generates Matplotlib charts.

### 02 — Weather ETL Pipeline
Scheduled pipeline that hits the OpenWeatherMap API every hour for 5 Indian cities. Parses JSON, deduplicates, and stores readings in PostgreSQL. Runs continuously with error handling and logging.
